# Comunidad WGH Argentina

Landing page de la **Comisión Comunidad** de Women in Global Health Argentina — espacio de las socias: encuentros, mentoría y vínculos.

Página HTML self-contained (un solo archivo, sin paso de build). Lista para publicar en GitHub Pages.
